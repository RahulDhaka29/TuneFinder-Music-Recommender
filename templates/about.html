About This Project

This Music Recommendation System is a machine learning project designed to help you discover new music based on your current tastes. It uses a **Content-Based Filtering** approach.

How It Works

Unlike systems that rely on what other users listen to, this recommender analyzes the intrinsic musical properties of the songs themselves. When you enter a song, the model looks at a vector of its audio features provided by Spotify, such as:

  • Danceability: How suitable a track is for dancing.
  • Energy: A perceptual measure of intensity and activity.
  • Valence: The musical positiveness conveyed by a track.
  • Tempo: The speed of the track in beats per minute (BPM).

It then uses a **Cosine Similarity** algorithm to find other songs in the dataset that have the most similar audio feature vectors.

Technology Stack

  • Backend: Python, Flask
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Frontend: HTML, Tailwind CSS
  • Data Source: Spotify Tracks Dataset from Kaggle
  • Live Data: Spotify Web API (for fetching album covers)
